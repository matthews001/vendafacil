# Rebranding — FechAí

A marca exibida no painel, vitrine, comprovantes, metadados, manifestos e PWA foi alterada de **VendaFácil** para **FechAí**.

## Compatibilidade preservada

Os identificadores técnicos já existentes que começam com `vendafacil:` foram mantidos de propósito. Eles armazenam preferências locais, sessões de portais e rascunhos no navegador. Alterá-los junto com a marca poderia apagar estados de usuários que já acessaram o sistema.

O cache do PWA foi renovado para `fechai-pwa-v1-brand`, fazendo os navegadores baixarem os novos arquivos de marca no próximo acesso.
