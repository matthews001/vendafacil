# VendaFácil — PDV Passo 5

Inclui o fluxo de venda de balcão: pagamento em dinheiro, Pix, débito, crédito, outro ou pendente; troco; desconto; registro seguro do pedido; baixa de estoque somente em venda paga; e comprovante simples para impressão.

## Antes de testar

1. Publique este projeto na Vercel.
2. No Supabase SQL Editor, execute `supabase/migrations/20260627_4_pdv_balcao_pagamento.sql`.
3. Abra a Frente de Caixa, monte o pedido e clique em **Finalizar venda e pagamento**.

Não precisa alterar as variáveis da Vercel para esta etapa.
